# Gate 7: 历史业绩合规门禁

## 触发条件

投资建议书或绩效报告中引用历史业绩时。

## 检查清单

- [ ] 每个历史业绩数据点旁标注"过往业绩不代表未来表现"
- [ ] 数据来源已注明
- [ ] 数据日期已标注
- [ ] 统计数据已注明样本量和统计期间

## 通过条件

所有历史业绩引用旁都有合规声明，且来源和日期明确。

## 如果失败

**BLOCKING ACTION:** 自动在每个历史业绩数据点旁追加合规声明。

## 强度

**HARD** — 自动修正后可通过，无需完全重写。

## 记录

```
{
  "gate": "gate-7-historical-performance",
  "status": "pass | auto-corrected",
  "data_points_checked": 5,
  "data_points_corrected": 1,
  "correction_made": "added compliance statement to '过去3年收益率15%'",
  "timestamp": "..."
}
```

## 合规声明模板

```
[产品名称] 过往业绩不代表未来表现，过往业绩不构成对未来收益的承诺。
数据来源：[XXX]
数据截至：XXXX年XX月
```

## 统计期间要求

| 引用类型 | 最低要求 |
|---------|---------|
| 基金净值 | 标注完整统计期间 |
| 业绩归因 | 说明样本量和归因期间 |
| 对比数据 | 说明基准和统计期间 |
