# 流失预警与挽回 (Churn Prevention)

## Overview

识别即将流失的客户，主动触达挽回，提升客户留存率。

**核心定位**：把"被动等客户来问"变成"主动发现预警信号并干预"。

## When to Use

- 定期扫描客户状态（建议每周一次）
- 客户沉默超期
- 收益率持续不达预期
- 客户表达不满/投诉

## 预警信号识别

### 红色预警（立即处理）

| 信号 | 定义 | 阈值 |
|------|------|------|
| 资金异动询问 | 客户主动询问"能取出多少钱" | 任意 |
| 投诉记录 | 客服/投诉系统记录 | 任意 |
| 30天沉默 | 没有任何互动 | 30天 |
| 收益严重不达 | 收益率低于基准 | >20% |

### 黄色预警（本周处理）

| 信号 | 定义 | 阈值 |
|------|------|------|
| 60天沉默 | 没有任何互动 | 60天 |
| 收益不达预期 | 收益率低于目标 | >10% |
| 转介绍率下降 | 原本转介绍的客户不再推荐 | 连续2次 |
| 沟通态度变化 | 回复变冷淡/延迟 | 连续3次 |

### 绿色观察（本月处理）

| 信号 | 定义 | 阈值 |
|------|------|------|
| 90天沉默 | 没有任何互动 | 90天 |
| 产品到期未续 | 封闭期产品到期无操作 | 任意 |
| 风险偏好变化 | 客户主动要求换高/低风险 | 任意 |

## 客户分层干预

### 第一梯队：高净值客户（≥1000万资产）

**策略**：高管/资深投顾优先回访

**干预流程**：
1. 识别预警信号 → 15分钟内升级
2. 资深投顾60分钟内回访
3. 了解不满原因
4. 制定个性化留存方案
5. 24小时内反馈处理结果

### 第二梯队：重点客户（100-1000万资产）

**策略**：主动关怀为主

**干预流程**：
1. 预警信号识别 → 当日分配
2. 投顾助理24小时内触达
3. 收集反馈
4. 必要时升级资深投顾介入

### 第三梯队：普通客户（<100万资产）

**策略**：标准化干预

**干预流程**：
1. 预警信号识别 → 批量处理
2. 自动触发关怀消息
3. 一周无响应 → 电话触达
4. 两周无响应 → 标记观察

## 流失原因分类

| 原因 | 表现 | 应对策略 |
|------|------|---------|
| 收益不达 | 投诉/沉默 | 归因分析 + 预期调整 + 调仓建议 |
| 服务不满 | 投诉/冷淡 | 服务复盘 + 升级处理 |
| 竞品吸引 | 转问别家 | 差异化价值说明 |
| 资金需求 | 取出资金 | 分批方案 + 留下部分 |
| 人生阶段 | 退休/移民 | 资产规划 + 长期关系 |

## 挽回话术模板

### 收益不达挽回

```
[客户姓名]您好，我是[投顾姓名]。

最近市场波动较大，我特别关注了您的持仓情况。

[坦诚说明]：目前组合收益[ X% ]，确实没有达到您预期的[ X% ]目标。

[分析原因]：主要是因为[市场因素/策略因素]，我们做了以下分析：[简述归因]

[下一步]：基于目前情况，我们建议[调整方案]。

我想约您通个电话，详细说说情况，您看明天下午方便吗？
```

### 沉默客户激活

```
[客户姓名]您好，我是[投顾姓名]。

近期市场[涨/跌/震荡]，我第一时间想到了您，想跟您同步一下市场情况。

[简述市场]：...[2-3句话]

您的持仓[表现良好/受到一些影响]，我这边有几个想法想跟您聊聊。

您看这周有时间吗？电话或见面都可以。
```

## 留存效果评估

| 指标 | 定义 | 目标 |
|------|------|------|
| 预警响应率 | 红色预警24小时内响应 | ≥90% |
| 挽回成功率 | 预警客户90天内无转出 | ≥70% |
| 沉默激活率 | 沉默客户激活回复 | ≥40% |
| 流失率 | 季度客户流失占比 | <5% |

## Example

**流失挽回示例：**

客户李女士，1000万资产，30天无互动，触发红色预警

**预警信号：**
- 30天无任何互动
- 近期收益率-8%，低于基准5%

**挽回处理：**

1. **识别（当日）：** 30天沉默触发红色预警，15分钟内升级
2. **分配：** 资深投顾60分钟内回访
3. **电话触达：**
```
"李女士您好，我是XX，很久没联系您了，特意打电话想跟您聊聊。

最近市场波动较大，我特别关注了您的持仓情况。目前收益确实不太理想，我想跟您详细说说原因...

（解释归因：主要是市场整体下跌，我们的回撤控制其实好于大盘...）

您看这周有时间吗？我想约您见面详细说说情况，同时也听听您的想法。"
```
4. **了解原因：** 客户表示"市场跌了也没人告诉我，感觉被忽略了"
5. **制定方案：** 安排每月主动报告一次 + 市场异动时即时触达
6. **结果：** 客户留存，继续持有
