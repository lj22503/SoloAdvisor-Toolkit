# 投资陪伴与市场沟通 (Investment Communication)

## Overview

在投后阶段与客户保持沟通，解释市场波动，管理客户预期和情绪。

**核心定位**：从"卖产品"到"卖服务"的关键环节。

## When to Use

- 市场发生波动时
- 定期报告发送后
- 客户主动来询时
- 持仓产品发生重大事件

## 沟通场景

### 场景一：定期报告解读

**时机**：周报/月报/季报发送后

**目标**：帮助客户理解报告内容

**要点**：
1. 先说结论，再解释原因
2. 用客户能理解的语言
3. 强调长期视角

**话术模板**：
```
"您上个月的组合收益是+2.3%，跑赢了基准1.5%。
主要是因为我们配置的债券基金表现不错，贡献了+1.8%。
股票部分有些波动，但整体在预期范围内。
您有什么想问的可以随时联系我。"
```

### 场景二：下跌安抚

**时机**：组合下跌时

**目标**：避免客户恐慌赎回

**原则**：
1. 共情先行，不轻视客户感受
2. 解释下跌原因（理性）
3. 回顾当初的配置逻辑
4. 提示不必此时赎回

**话术模板**：
```
"看到账户缩水，您一定不太舒服，这很正常。
这次下跌主要是市场整体调整，我们的组合回撤了X%。
从基本面看，经济并没有那么差，这次更多是情绪驱动。
您当初配置债券的比例是X%，这在下跌中起到了一定的缓冲作用。
目前不需要做什么操作，保持定投即可。"
```

### 场景三：上涨提醒

**时机**：组合上涨时

**目标**：防止客户盲目乐观

**原则**：
1. 肯定收益，但不夸大
2. 提醒风险，提醒目标非承诺
3. 讨论是否需要止盈

**话术模板**：
```
"恭喜您，组合这个月表现不错，涨了+5%。
但我也要提醒您，过往业绩不代表未来。
如果您当初设定的目标是年化8%，目前已经完成了X%。
您可以思考一下，是想继续持有还是部分止盈？
我这边可以给您几个方案供选择。"
```

### 场景四：产品/基金经理变更

**时机**：持仓产品发生重大事件

**目标**：解释影响，决定是否调整

**要点**：
1. 说明发生了什么
2. 评估影响程度
3. 说明我们的应对

**话术模板**：
```
"通知您一个重要信息，您持有的XX基金更换了基金经理。
新基金经理是XXX，有X年经验，擅长XXX风格。
我们需要观察一段时间来评估是否需要调整持仓。
目前我会密切关注基金的表现，如果出现异常会第一时间通知您。"
```

## 沟通频率建议

| 客户类型 | 主动沟通 | 被动响应 |
|---------|---------|---------|
| 新客户（<6月） | 每月1次 | 即时 |
| 稳健型 | 每季1次 | 即时 |
| 成长型 | 每季1次 | 即时 |
| 高净值 | 按需+每季 | 即时 |
| 预警客户 | 每月2次 | 即时 |

## 沟通渠道选择

| 渠道 | 适用场景 | 注意事项 |
|------|---------|---------|
| 电话 | 重要事项、情绪安抚 | 记录要点 |
| 微信/短信 | 日常问候、报告发送 | 保持专业 |
| 面访 | 年度复盘、重大决策 | 提前准备 |
| 报告 | 信息同步 | 配套解读 |

## 投资者教育

### 常见误区

| 误区 | 正确认知 |
|------|---------|
| "基金要长期持有就不管了" | 需要定期检视 |
| "亏了就赎回来止损" | 可能是暂时性亏损 |
| "涨得好就要加仓" | 可能已高估 |
| "分散投资就是买很多基金" | 需要相关性低 |

### 教育话术

```
"基金投资有几个要点您可以记住：
1. 收益和风险是匹配的，高收益必然伴随高波动
2. 短期涨跌很正常，不要太在意
3. 适当远离市场，反而更容易赚钱
4. 定期检视是必要的，但不要频繁操作"
```

## Example

**下跌安抚话术示例：**

市场下跌日，客户张先生组合回撤-3.2%

```
"张先生，看到账户缩水，您一定不舒服，这很正常。

这次下跌主要是市场整体调整，我们的组合回撤了3.2%。

从基本面看，经济并没有那么差，这次更多是情绪驱动。

您当初配置了40%的债券，在下跌中起到了一定的缓冲作用。

目前不需要做什么操作，保持定投即可。

我会密切关注情况，有变化第一时间通知您。"
```
