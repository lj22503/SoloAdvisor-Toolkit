# 养老金/教育金规划 (Pension & Education Planning)

## Overview

针对养老和教育金等特定财务目标，进行目标现值法测算和定投规划。

**核心定位**：把长期财务目标量化，提供可执行的储蓄/投资方案。

## When to Use

- 客户有明确养老/教育金需求
- KYC中发现相关目标
- 定期检视时

## 核心方法

### 目标现值法

计算实现目标所需的当前投入金额。

```
PV = FV / (1 + r)^n

PV = 现值（当前需要准备的钱）
FV = 终值（目标金额）
r = 年化收益率
n = 年数
```

### 定期定投法

计算每月需要投入的金额。

```
PMT = PV × r / [1 - (1 + r)^(-n)]
```

## 养老规划

### 参数设定

| 参数 | 常见值 | 说明 |
|------|--------|------|
| 退休年龄 | 60岁 | 可调整 |
| 预期寿命 | 85岁 | 按需调整 |
| 退休后年收入替代率 | 60-70% | 退休前年收入的百分比 |
| 社保替代率 | 30-40% | 估算社保养老金 |
| 预期年化收益率 | 6-8% | 权益资产配置比例决定 |

### 测算示例

```
客户：张先生，45岁，税后年收入50万
目标：60岁退休，退休后年收入40万（80%替代率）
社保估算：月领5000，年6万
退休缺口：40万 - 6万 = 34万/年

目标终值FV = 34万 × 25年（85-60）= 850万
假设年化收益6%：
当前现值PV = 850万 / (1.06)^15 ≈ 355万
从现在起每月定投 ≈ 355万 × 6% / [12 × (1 - 1.06^(-180))] ≈ 2.3万/月
```

## 教育金规划

### 参数设定

| 参数 | 常见值 | 说明 |
|------|--------|------|
| 子女当前年龄 | 0-18岁 | 距使用年限 |
| 教育阶段 | 大学/研究生/全程 | 目标不同 |
| 目标金额 | 50-200万 | 按需调整 |
| 预期年化收益率 | 5-7% | 偏保守配置 |

### 教育金测算示例

```
客户：李女士，子女5岁，计划18岁上大学
目标：届时准备100万教育金
距使用：13年
假设年化收益6%：
当前现值PV = 100万 / (1.06)^13 ≈ 48万
每月定投 ≈ 48万 × 6% / [12 × (1 - 1.06^(-156))] ≈ 0.3万/月
```

## 产品推荐框架

| 目标期限 | 推荐产品 | 风险等级 | 配置比例 |
|---------|---------|---------|---------|
| >20年 | 股票型基金定投 | R4-R5 | 60-80% |
| 10-20年 | 混合型基金 | R3-R4 | 40-60% |
| 5-10年 | 债基+混基 | R2-R3 | 30-50% |
| <5年 | 短债+货基 | R1-R2 | 80-100% |

## 合规要求

- 不得承诺收益
- 风险揭示必须充分
- 历史业绩必须标注"过往不代表未来"

## Example

**养老规划示例：**

张先生，45岁，税后年收入50万，计划60岁退休
- 目标：退休后年收入40万
- 社保估算：年6万
- 退休缺口：34万/年
- 距离退休：15年

**测算结果：**
- 目标终值：850万（34万×25年）
- 当前需准备：355万
- 每月定投：约2.3万
- 建议配置：养老目标基金60% + 定期年金40%
