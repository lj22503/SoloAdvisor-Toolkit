# 绩效回顾与报告 (Performance Review)

## Overview

定期回顾组合表现，进行归因分析，生成客户报告。

**核心定位**：让客户理解收益来源，管理预期。

## When to Use

- 定期报告（周/月/季/年）
- 客户问询时
- 重大事件后

## 归因分析框架

### Brinson 模型

组合收益分解：

```
总收益 = 资产配置收益 + 产品选择收益 + 交互效应

资产配置收益 = (策略配置比例 - 基准配置比例) × 基准资产收益
产品选择收益 = 策略配置比例 × (实际产品收益 - 基准资产收益)
交互效应 = (策略-基准) × (实际-基准)
```

### 收益归因表

| 贡献来源 | 贡献金额 | 贡献占比 |
|---------|---------|---------|
| 资产配置 | +X% | XX% |
| 产品选择 | +X% | XX% |
| 交互效应 | -X% | XX% |
| 费用影响 | -X% | XX% |
| **合计** | **+X%** | **100%** |

## 评估指标

### 收益指标

| 指标 | 计算 | 解读 |
|------|------|------|
| 持有期收益 | 期末净值/期初净值 - 1 | 绝对收益 |
| 年化收益 | (1+持有收益)^(365/持有天数) - 1 | 标准化 |
| vs 基准 | 组合收益 - 基准收益 | 相对收益 |

### 风险指标

| 指标 | 计算 | 解读 |
|------|------|------|
| 最大回撤 | 历史最大跌幅 | 下行风险 |
| 年化波动率 | 收益标准差年化 | 波动水平 |
| 夏普比率 | (收益-无风险)/波动率 | 风险调整收益 |
| 卡玛比率 | 年化收益/最大回撤 | 综合评价 |

### 客户体验指标

| 指标 | 含义 |
|------|------|
| 正收益月份占比 | 持有体验 |
| 回本时间 | 心理账户 |
| 最大浮亏感受 | 焦虑程度 |

## 沟通要点提炼

归因后，需要提炼给客户的核心信息：

| 收益来源 | 客户能理解的说法 |
|---------|---------------|
| 股票仓位低 | "我们低配了股票，避开了下跌" |
| 债券正贡献 | "债券仓位提供了稳定收益" |
| 产品选择好 | "选到的基金跑赢了同类" |
| 费用拖累 | "费用是收益的正常损耗" |

## 合规要求

- 历史业绩必须标注"过往不代表未来"
- 归因数据必须来源可靠
- 报告需存档

## Example

**季度回顾示例：**

张先生，2025年Q2

**收益情况：**
- 季度收益：+5.2%
- 基准收益：+4.1%
- 跑赢基准：+1.1%

**归因分析：**
| 来源 | 贡献 |
|------|------|
| 股票仓位（+10%） | +1.8% |
| 债券仓位（稳定） | +0.5% |
| 产品选择 | +0.8% |
| 费用 | -0.4% |
| 合计 | +2.7% |

**给客户的核心话术：**
> "您上季度收益+5.2%，跑赢了基准1.1个百分点。主要是因为我们配置的股票基金表现不错。如果市场下跌，我们的债券仓位能起到一定的缓冲作用。"
