# 组合构建 (Portfolio Construction)

## Overview

基于资产配置方案，将具体产品落地，构建完整投资组合。

**核心定位**：把"买什么、买多少"变成可执行的组合方案。

## When to Use

- 资产配置方案审批后
- 再平衡触发时
- 持仓检视时

## 组合构建流程

```
第一步：确定配置比例
  → 根据SAA确定各大类比例
  → 根据TAA进行微调

第二步：产品匹配
  → 从基金筛选结果中匹配产品
  → 确保风格互补
  → 避免重复持仓

第三步：建仓规划
  → 分批还是一次性
  → 建仓节奏
  → 成本估算

第四步：风险验证
  → 集中度检查（Gate 4）
  → 适当性检查（Gate 2）
  → 情景分析
```

## 组合构建原则

### 核心-卫星策略

| 层级 | 比例 | 特点 | 产品类型 |
|------|------|------|---------|
| 核心 | 60-80% | 稳定、被动型 | 指数基金、大盘蓝筹 |
| 卫星 | 20-40% | 主动、机会型 | 行业主题、择时操作 |

### 分散化原则

| 维度 | 分散要求 |
|------|---------|
| 资产类别 | 股/债/商品至少2类 |
| 地域 | 单一市场<60% |
| 风格 | 价值/成长均衡 |
| 期限 | 短中长期搭配 |

## 权重分配

### 等权重法

每个产品权重相等，适合无明确观点时。

```
权重 = 1 / N
N = 产品数量
```

### 风险平价法

各资产对组合风险的贡献相等。

```
风险贡献 = 权重 × 波动率 / 组合波动率
目标：各资产风险贡献相等
```

### 最小方差法

在给定预期收益下，组合方差最小。

```
min σ²_p = w'Σw
s.t. w'μ = μ_p
     Σw = 1
     w ≥ 0
```

## 再平衡触发条件

| 条件 | 动作 |
|------|------|
| 偏离配置>5% | 预警 |
| 偏离配置>10% | 执行再平衡 |
| 年度检视 | 全面评估 |
| 重大市场变化 | 临时检视 |

## 情景分析

构建组合后，进行压力测试：

| 情景 | 冲击 | 组合预期回撤 |
|------|------|-------------|
| 市场下跌20% | 沪深300 -20% | -8% ~ -12% |
| 利率上行1% | 债券 -5% | -2% ~ -4% |
| 风格切换 | 成长→价值 | -5% ~ -8% |
| 流动性危机 | 全面-30% | -15% ~ -20% |

## 合规要求

- 适当性匹配确认（Gate 2）
- 集中度检查（Gate 4）
- 风险揭示完整（Gate 3）
- 费用披露完整（Gate 6）

## Example

**组合构建示例：**

张先生，3000万，C3客户

**配置方案：**
| 资产 | 产品 | 比例 | 金额 |
|------|------|------|------|
| 国内股票 | XX沪深300 | 25% | 750万 |
| 国内股票 | XX成长股票 | 10% | 300万 |
| 债券 | XX纯债基金 | 40% | 1200万 |
| 商品 | XX黄金ETF | 5% | 150万 |
| 现金 | 货币基金 | 20% | 600万 |

**建仓规划：**
- 方式：分批建仓，3个月6批
- 首批：30%（900万）
- 留20%现金应对波动

**再平衡规则：**
- 偏离>10%时触发再平衡
- 每季度全面检视
