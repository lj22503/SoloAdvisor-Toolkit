# 私募/信托产品评估 (Private Fund Evaluation)

## Overview

评估私募基金、信托计划等非公募产品的投资价值与风险。

**核心定位**：高净值客户专属产品线的风险评估。

## When to Use

- 高净值客户有私募配置需求
- 信托产品筛选
- 私募代销准入评估

## 产品分类

| 类型 | 风险等级 | 投资门槛 | 锁定期 |
|------|---------|---------|--------|
| 私募证券基金 | R3-R5 | 100万 | 6-12月 |
| 私募股权基金 | R5 | 100万+ | 3-5年 |
| 信托计划 | R2-R4 | 100万+ | 1-3年 |
| 家族信托 | R2-R3 | 1000万+ | 长期 |

## 评估维度

### 投资策略

| 维度 | 评估内容 | 重要性 |
|------|---------|--------|
| 策略类型 | 主观多头/量化/CTA/固收+ | ★★★ |
| 策略容量 | 规模上限是否匹配 | ★★★ |
| 策略相关性 | 与现有组合的相关性 | ★★ |
| 策略有效期 | 策略适应性周期 | ★★ |

### 业绩评估

| 指标 | 计算 | 解读 |
|------|------|------|
| 年化收益 | IRR / 持有年限 | vs 基准 vs 同策略 |
| 最大回撤 | 历史最大跌幅 | 绝对风险 |
| 胜率 | 正收益月份占比 | 稳定性 |
| 夏普比率 | (收益-无风险)/波动率 | 风险调整收益 |
| 卡玛比率 | 年化收益/最大回撤 | 综合评价 |

### 团队评估

| 维度 | 评估内容 |
|------|---------|
| 核心人物 | 从业年限、历史业绩、稳定性 |
| 团队结构 | 投研人数、背景互补性 |
| 激励机制 | 跟投机制、奖金递延 |

### 条款分析

| 条款 | 风险点 |
|------|--------|
| 管理费 | 高管理费侵蚀收益 |
| 业绩报酬 | 提成方式（高水位/rrh） |
| 申赎条款 | 锁定期、赎回费 |
| 预警止损 | 预警线/止损线设置 |
| 流动性安排 | 特殊情况下赎回条款 |

## 风险揭示重点

| 产品类型 | 特有风险 |
|---------|---------|
| 私募证券 | 流动性差、净值披露不及时 |
| 私募股权 | 周期长、估值困难 |
| 信托 | 信用风险、项目风险 |
| 结构化 | 杠杆风险、优先劣后结构 |

## 合规要求

- 必须为合格投资者
- 风险揭示书必须详细
- 冷静期确认（24小时）
- 适当性匹配确认

## Example

**私募评估示例：**

张先生，C4客户，可投资资产5000万，配置15%私募

**候选产品：**
- 产品：XX私募证券基金
- 策略：主观多头
- 规模：30亿

**评分：**
| 维度 | 得分 | 说明 |
|------|------|------|
| 策略 | 4.0 | 策略容量50亿，当前30亿，合适 |
| 业绩 | 4.5 | 3年年化18.5%，回撤-22% |
| 团队 | 4.0 | 核心人物10年经验 |
| 条款 | 3.5 | 管理费1.5%，业绩20%高水位 |

**结论：** 综合评分4.0，适合C4客户，配置比例建议≤15%
