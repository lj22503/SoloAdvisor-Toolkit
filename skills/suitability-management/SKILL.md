# 适当性管理 (Suitability Management)

## Overview

确保产品风险等级与客户风险等级匹配，管理系统性适当性风险。

**核心定位**：合规核心，监管红线。

## When to Use

- 组合构建完成后
- 产品推荐前
- 客户风险等级变更时
- 定期适当性检视

## 适当性矩阵

### 风险等级对照

| 客户风险等级 | 可购买产品风险等级 | 最高配置比例 |
|-------------|------------------|-------------|
| C1（保守型） | R1 | 100% |
| C2（稳健型） | R1, R2 | R2≤30% |
| C3（平衡型） | R1, R2, R3 | R3≤20% |
| C4（成长型） | R1, R2, R3, R4 | R4≤30% |
| C5（进取型） | R1-R5 | R5≤50% |

### 产品风险等级

| 产品类型 | 风险等级 | 典型回撤 |
|---------|---------|---------|
| 货币基金 | R1 | 0% |
| 短债基金 | R2 | <2% |
| 纯债基金 | R2 | <5% |
| 二级债基 | R3 | <10% |
| 混合基金 | R3-R4 | <20% |
| 股票型基金 | R4 | <30% |
| 私募/结构化 | R5 | >30% |

## 适当性检查流程

```
┌─────────────────────────────────────────────────────────┐
│  Step 1: 确认客户风险等级                               │
│    → 读取KYC档案中的风险等级                           │
│    → 确认是否在有效期内（1年）                         │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  Step 2: 确认产品风险等级                               │
│    → 查询产品数据库                                     │
│    → 确认产品风险等级                                   │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  Step 3: 适当性匹配判断                                 │
│    → 产品风险 ≤ 客户风险？                              │
│    → 比例超限？                                         │
└─────────────────────────────────────────────────────────┘
                          ↓
                   匹配检查结果
                          ↓
┌────────────┐                        ┌────────────────┐
│   通过     │                        │   不通过       │
│            │                        │                │
│ → 记录通过 │                        │ → 拦截/降级   │
│ → 继续流程 │                        │ → 记录原因     │
└────────────┘                        └────────────────┘
```

## 不适当销售处理

### 情形一：产品风险 > 客户风险

**处理**：禁止推荐

```
不允许推荐R4产品给C3客户
建议方案：
1. 降级产品（R3替代R4）
2. 减少比例（R4配置≤10%）
3. 拒绝推荐
```

### 情形二：产品风险 = 客户风险上限

**处理**：限制比例，额外揭示

```
C3客户购买R3产品接近上限
要求：
1. 配置比例≤20%
2. 客户签署"风险确认书"
3. 投顾签字确认
```

### 情形三：客户风险等级过期

**处理**：重新测评

```
风险测评有效期1年
超期处理：
1. 提示客户重新测评
2. 暂停新推荐
3. 持仓不受影响
```

## 定期检视

| 检视频率 | 内容 |
|---------|------|
| 持仓后30天 | 首次适当性确认 |
| 每季度 | 持仓适当性检视 |
| 客户风险变更时 | 立即检视 |
| 产品风险变更时 | 立即检视 |

## 合规要求

- 必须留存检查记录
- 不适当销售必须上报
- 客户确认书必须存档
- 投顾签字确认

## Example

**适当性检查示例：**

张先生，C3客户，申请购买XX成长股票基金（R4）

**检查结果：**
```
产品风险：R4
客户风险：C3
匹配结果：FAIL

原因：R4产品风险超过C3客户风险上限

建议方案：
1. 降级至R3产品（如XX平衡混合基金）
2. 或减少R4产品比例至≤20%
3. 或拒绝本次购买
```

**如客户坚持购买R4：**
- 需CEO审批
- 客户签署"超风险确认书"
- 记录豁免原因
