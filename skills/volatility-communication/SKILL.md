# 波动阶段市场沟通 (Volatility Communication)

## Overview

在市场波动时，主动触达客户，解释市场变化，安抚情绪，管理预期。

**核心定位**：以"市场热点"为敲门砖，系统回答投后三问（沟通频率/收益预期/策略应对）。

## When to Use

- 市场单日涨跌幅超3%
- 持仓产品发生重大事件
- 客户来询（主动询问市场）
- 定期市场回顾（每周/每月）

## 工作流程

### 第一步：市场热点解读

**分析框架**（借鉴 b-event 三把尺子）：

| 维度 | 评分（1-10） | 说明 |
|------|-------------|------|
| 不可逆性 | X | 事件是否不可逆？政策放开通常不可逆 |
| 影响半径 | X | 影响范围多大？ |
| 认知时差 | X | 专业 vs 散户认知差 |

**事件分类**：
- 闸门事件：规则改变
- 管道事件：渠道疏通
- 背离事件：认知差

**四步心法**：
1. 望气：宏观信号判断
2. 走卒：资金流向判断
3. 实业：实体经济影响
4. 操盘：实际操作建议

### 第二步：确定沟通频率

基于客户画像、持仓状态、市场阶段，判断沟通频率。

| 市场阶段 | 客户持仓状态 | 建议沟通频率 |
|---------|------------|-------------|
| 大幅上涨 | 盈利中 | 电话告知，关注止盈意愿 |
| 大幅下跌 | 亏损中 | 当日主动触达，安抚为主 |
| 震荡 | 盈亏平衡 | 陪伴报告即可 |
| 重大事件 | 任意 | 事件驱动，即时触达 |

### 第三步：评估收益预期

**分析框架**：
1. 明确产品属性（固收+/权益/FOF/指数/行业主题）
2. 设定合理收益区间（绝对收益目标 or 相对基准）
3. 当前收益 vs 满意区间
4. 预期修复时间

**沟通原则**：
- 已达到：提醒非承诺，建议理性看待
- 未达到：解释原因，给出预期
- 严重亏损：共情优先，不轻言"会涨回来"

### 第四步：明确策略及变化应对

**三问回答**：
1. **沟通频率**：这次沟通后，多久再联系？
2. **收益预期**：目前收益是否在预期内？
3. **策略应对**：当前策略是什么？变了怎么办？

**触发条件**（提前告知客户）：
- 条件1：如"沪深300估值突破历史80%分位"
- 条件2：如"产品回撤超过X%"
- 条件3：如"基金经理风格显著漂移"

## 沟通场景模板

### 上涨场景

```
【市场上涨沟通模板】

标题：市场上涨，您的持仓表现良好

今日市场解读：
[市场热点 + 三把尺子评分]

您的持仓状态：
- 组合今日涨跌：+X%
- 成立以来收益：X%
- vs 基准：跑赢/跑输 X%

三问解答：
1. 多久沟通一次？
   → 建议[每X周]定期沟通一次，市场大幅波动时主动触达

2. 收益满意吗？
   → 目前收益[X%]在预期[X%-X%]范围内...
   → 注意：过往业绩不代表未来

3. 策略会变吗？
   → 当前策略：[简述]
   → 触发调整条件：[已告知客户]
   → 如果调整：[将主动联系您]
```

### 下跌场景

```
【市场下跌沟通模板】

标题：市场波动，您的持仓说明

今日市场解读：
[市场热点 + 三把尺子评分 + 重点：这不是恐慌的理由]

您的持仓状态：
- 组合今日涨跌：-X%
- 最大回撤：-X%
- vs 基准：跑赢/跑输 X%

三问解答：
1. 多久沟通一次？
   → 我们会[本周内]主动联系您，了解您的想法

2. 亏损怎么办？
   → 当前属于[正常波动/超出预期]
   → 原因分析：[解释下跌原因]
   → 预期修复：[时间估算]

3. 策略会变吗？
   → 当前策略[不变，坚持理由] / [有调整必要，调整方向]
   → 如果调整：[主动联系说明]
```

## 响应时效

| 场景 | 触发条件 | 响应时效 |
|------|---------|---------|
| 紧急安抚 | 跌超5%或重大事件 | 当日 |
| 主动告知 | 涨/跌超3% | T+1 |
| 定期陪伴 | 每周/每月 | 按计划 |

## Example

**市场下跌沟通示例：**

今日市场下跌-4.5%，持仓客户张先生（亏损中）

**市场解读：**
- 事件：三把尺子评分（不可逆性7/影响半径8/认知时差6）
- 类型：闸门事件，非恐慌性质
- 原因：情绪驱动而非基本面恶化

**三问解答：**
1. 多久沟通？ → 本周内电话主动告知
2. 亏损怎么办？ → 解释原因（情绪驱动），给出预期修复时间
3. 策略变吗？ → 条件未触发，策略不变

**话术：**
> "张先生，今天市场跌了不少，您的组合估计也缩水了，我第一时间想到要跟您说一下。

这次下跌主要是情绪驱动，不是基本面恶化。我们配置的债券仓位起到了一定的缓冲作用。

目前策略不需要调整，如果市场继续下跌超过X%，我会提前跟您沟通。

您不用太担心，保持正常心态就好。"

## 合规要求

- 所有沟通必须留存记录
- 不得承诺收益
- 不得暗示"保本"
- 历史业绩必须标注"过往不代表未来"
